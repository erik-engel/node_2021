NPM


Express


Express is a framework that's running within node. We use it for building webapplications quickly.

Express is installed through the npm (node package manager) let's see how:


{
     "dependencies": {
        "cross-env": "^7.0.3",
         "express": "^4.17.1"
    }
}

The above example is how you should configure your package.json. Now you just type npm install in your terminal, of course in the right location. Now express will be install and you're ready to use it.


Another way to install express would be by typing "npm install express" in the terminal.


Nodemon


Nodemon is a great tool we can use for developing webapps. Nodemon will restart the application whenever a change is made


We can install nodemon the same way we would with express. Yet here comes a different way to install it, which you propaply better like in the long run.


This will install nodemon globally


npm install -g nodemon

Now, insted of the usual way to run a node file, we can type the following:


npm install -g nodemon

Now the application will restart whenever a change is made to the files.


With express and nodemon both installed, we are now ready to create an API.
