Functions & Callback Functions


JavaScript Functions


We use functions in JavaScript. That's a fact!


So in JS we use regular functions, variable functions and arrow functions. Lets have an example of all.


//regular function
function multiply(a, b) {
     return a*b
}

multiply(2,4) //output = 8

//variable function
const greeting = function () {
     console.log('hello there') }

greeting() //output = 'hello there'

//arrow function
const greetingAswell = () => {
     console.log('hello there aswell')
}

greetingAswell() //output = 'hello there aswell'

So the first example just above is a function decalartion


In the second example greeting isn't actually a function itself. It's a statement. The function itself is what we call an anonymous function which is saved to a variable. In our case the variable is called greeting.


The arrow function does the same thing as the it an anonymous function saved to a variable. In our case grettingsAswell. The reason we prefer arrow functions over normal functions is that they declare the 'this' keyword to the scope of the function, where in normal variable function, the 'this' keyword would depend on where it was called.



Callbacks


A callback function is simply put a function that is passed to a function as a parameter ..


Lets have an example:


function callStatement(functionRef) {
     functionRef();
}
const statement = () => {
     console.log('This is a callback!')
}
callStatement(statement); //output "This is a callback!

So in the above example we call callStatement so it states 'This is a callback' which comes from the callback function statement()
